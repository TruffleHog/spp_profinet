\subsection{Detailed Description}
The sender interface. The basic \hyperlink{struct_sender}{Sender} abstraction. Every implementation of a \hyperlink{struct_sender}{Sender} will use and implement the operations described in this interface. A \hyperlink{struct_sender}{Sender} is used to send truffles to a certain port, socket, or messagequeue, depending on the implementation.
